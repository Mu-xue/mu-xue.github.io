Vue + Vue Loader + Mint-ui 搭建的场馆预定系统Demo

发表于 2018-07-07 | 分类于 个人作品

仿运动猿公众号的一个场馆预定移动端网站

刚好前段时间学了Vue，有了React的基础感觉这个框架上手起来真的比较简单，于是趁着学校一门课也得做个小项目，正好就来弄这么个东西吧~

搭建步骤：

  1. 利用vue-cli新建项目
  2. 配置好 mint-ui
  3. 设计好组件架构。这一步对于一个项目来说是很重要的，关系到后面开发时代码耦合性及可重用性。以这个项目的首页为例：

我简单地将它分为四个部分：头部的Nav组件，中部的Sport组件，底部的GymList组件，由于GymItem复用性很高，我把GymList再拆分成Tab和GymItem，这样开发上结构也比较清晰，容易维护。

  1. 接下来就是愉快的开发时间了~
